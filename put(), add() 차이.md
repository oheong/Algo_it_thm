# put(), add() 차이



문득 오늘 문제풀다가,,,, put()이랑 add()가 너무 헷갈리는거임,,,,



지금 당장 생각나는 메소드 별 자바 클렉션들은

##### add() => LinkedList(Queue), Stack, ArrayList, 기타등등,,,,,

##### put() => HashMap, 등등,,,,,,,,

이것도 생각나면 더 추가해놓겠음



여튼 서칭 좀 해 본 결과

- 공통점 : List, Map등 자료구조에서 데이터를 추가한다.
- put() : 자료구조 내부에서 위치가 어디든 상관 ❌. 순서가 없는경우
- add() : 자료구조 내부에서 순서가 존재하는 경우
